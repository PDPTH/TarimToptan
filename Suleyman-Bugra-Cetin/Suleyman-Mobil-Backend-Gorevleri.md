# Süleyman Buğra Çetin - Mobil Back-End Görevleri

## Sorumlu REST API Bağlantıları

1. `POST /auth/register`
2. `POST /auth/login`
3. `PUT /users/{userId}`
4. `DELETE /users/{userId}`
5. `GET /producers`
6. `GET /producers/{producerId}`
7. `POST /auth/reset-password-request`

## Kod Konumu

- Mobil bağlantı kodu: `android-java/app/src/main/java/com/tarimtoptan/android/features/account/AccountFeature.java`
- Ortak API istemcisi: `android-java/app/src/main/java/com/tarimtoptan/android/net/ApiClient.java`
