# Umut Can Bayar - Mobil Back-End Görevleri

## Sorumlu REST API Bağlantıları

1. `POST /products`
2. `PUT /products/{productId}`
3. `DELETE /products/{productId}`
4. `GET /products`
5. `GET /products/{productId}`
6. `PATCH /products/{productId}/stock`
7. `POST /products/{productId}/reviews`

## Kod Konumu

- Mobil bağlantı kodu: `android-java/app/src/main/java/com/tarimtoptan/android/features/products/ProductsFeature.java`
- Ortak API istemcisi: `android-java/app/src/main/java/com/tarimtoptan/android/net/ApiClient.java`
