# Aykhan Bayramov - Mobil Back-End Görevleri

## Sorumlu REST API Bağlantıları

1. `POST /cart/items`
2. `DELETE /cart/items/{itemId}`
3. `GET /cart`
4. `POST /orders`
5. `DELETE /orders/{orderId}`
6. `POST /addresses`
7. `PUT /addresses/{addressId}`

## Kod Konumu

- Mobil bağlantı kodu: `android-java/app/src/main/java/com/tarimtoptan/android/features/checkout/CheckoutFeature.java`
- Ortak API istemcisi: `android-java/app/src/main/java/com/tarimtoptan/android/net/ApiClient.java`
